CoypuIDE is a set of widgets for the Coypu and Phausto music and DSP libraries, implemented in Bloc and Toplo. 

### Loading latest development version

```Smalltalk
Metacello new
	baseline: 'CoypuIDE';
	repository: 'github://pharo-graphics/CoypuIDE:master';
	load.
 ```
